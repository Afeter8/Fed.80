Ayuda rápida

• Esta POC ejecuta todo en el navegador. Para producción: usar Vault/KMS para ROT_KEY, mover rotator a servidores seguros, anclar manifest en IPFS/blockchain y segregar responsabilidades.
• No ejecute código desrotado en hosts de producción sin sandbox (contenedor, AppArmor/seccomp).
• Para integrar con GitHub real o ClouDNS real, usa sus APIs oficiales y guarda credenciales en un almacén seguro.
